# Student Performance Exploratory Data Analysis (EDA)

## Overview

This project performs **Exploratory Data Analysis (EDA)** on the **Students Performance** dataset using Python. The objective is to understand the dataset, identify patterns, analyze student performance, and visualize key insights using Pandas and Matplotlib.

The project demonstrates the complete EDA workflow, including data exploration, cleaning, feature engineering, statistical analysis, and data visualization.


## Objectives

- Load and inspect the dataset
- Explore the structure and quality of the data
- Analyze student performance across different categories
- Create new features for deeper analysis
- Visualize trends and relationships
- Draw meaningful insights from the data

## Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib


## Dataset

**Students Performance Dataset**

The dataset contains information about students, including:

- Gender
- Race/Ethnicity
- Parental Level of Education
- Lunch Type
- Test Preparation Course
- Math Score
- Reading Score
- Writing Score

## Exploratory Data Analysis Performed

### Data Exploration

- Loaded the dataset
- Examined dataset dimensions
- Inspected column names and data types
- Checked for missing values
- Generated descriptive statistics

### Data Analysis

- Gender distribution
- Race/Ethnicity distribution
- Test preparation completion
- Average subject scores
- Performance comparisons across different groups
- Student filtering based on various conditions
- Sorting and ranking students

### Feature Engineering

- Created an **Average Score** column
- Created **Pass/Fail** classification
- Assigned student **Grades**

### Group Analysis

- Average scores by gender
- Average scores by parental education
- Average scores by lunch type
- Average scores by test preparation course

### Data Visualization

- Bar Charts
- Histograms
- Pie Charts
- Scatter Plot
